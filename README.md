This repository contains a Power BI dashboard designed to analyze and visualize BMW sales performance. The dashboard provides an interactive way to explore sales data, track key performance indicators (KPIs), and gain insights into trends across different regions, models, and time periods.
Features
- 🚗 Sales Analysis by Model – Compare performance across BMW car models.
- 🌍 Regional Insights – View sales distribution across different geographies.
- 📈 Trend Tracking – Monitor monthly and yearly sales growth patterns.
- 🎯 KPIs & Metrics – Key indicators such as revenue, units sold, and market share.
- 🔍 Interactive Filters – Drill down into specific segments for deeper analysis.
